## Install spack:

<pre>git clone https://github.com/spack/spack.git
cd spack/bin
./spack install zlib
</pre>

## Install compilers:

<pre>sudo yum install gcc-gfortran</pre>

## Install scr (make sure you're in the spack/bin directory):

<pre>./spack install scr</pre>
